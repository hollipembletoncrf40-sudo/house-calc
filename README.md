# 买房投资决策计算器

一个专业的房地产投资分析工具，帮助您计算投资回报率、评估风险、做出明智的买房决策。

## 🌐 在线访问

**在线地址**: https://hollipembletoncrf40-sudo.github.io/house-calc/

**GitHub 仓库**: https://github.com/hollipembletoncrf40-sudo/house-calc

## 功能特性

### 💰 投资收益计算
- **现金回报率**：基于首付计算年租金净收入的回报率
- **综合回报率**：包含房价增值和租金收入的总投资回报
- **年化收益率**：标准化的投资绩效指标
- **月供覆盖比**：评估租金覆盖月供的能力

### ⚠️ 风险评估系统
- **现金流风险**：评估月供覆盖比、紧急备用金、收入负债比
- **杠杆风险**：评估首付比例和负债率
- **市场风险**：评估房价波动承受能力和预期涨幅合理性
- **综合风险评分**：0-100分制，直观展示投资风险等级

### 🔬 压力测试
模拟不利情景下的投资表现：
- 房价下跌10%/20%情景
- 租金下降20%情景
- 利率上升2%情景

### 💡 智能决策建议
基于计算结果和风险评估，自动生成：
- 总体评价和建议
- 收益分析
- 风险提示
- 具体行动建议

### 📈 可视化分析
使用Chart.js绘制收益趋势图，直观展示：
- 累计租金收益
- 房价增值
- 总收益变化

## 使用方法

### 1. 打开应用
访问在线地址或在浏览器中打开 `index.html` 文件。

### 2. 输入参数

#### 房屋信息
- **房屋总价**：目标房产的总价（万元）
- **首付比例**：计划支付的首付比例（%）
- **持有年限**：计划持有房产的年限

#### 贷款信息
- **贷款年限**：贷款期限（年）
- **贷款年利率**：当前贷款利率（%）

#### 租金与收益
- **月租金**：预期月租金收入（元）
- **持有成本比例**：物业费、维修、空置等成本占租金比例（%）
- **预期年涨幅**：预期房价年涨幅（%）

#### 风险评估
- **紧急备用金**：可用于应急的资金（万元）
- **家庭月收入**：家庭总月收入（元）

### 3. 查看结果
点击"开始计算"按钮，系统将自动计算并展示：
- 详细的收益指标
- 风险评估报告
- 压力测试结果
- 决策建议
- 可视化图表

## 计算公式说明

### 月供计算（等额本息）
```
月供 = 贷款本金 × 月利率 × (1 + 月利率)^还款月数 / [(1 + 月利率)^还款月数 - 1]
```

### 现金回报率
```
年租金净收入 = 年租金 - 年持有成本（租金的15%）
现金回报率 = 年租金净收入 / 首付款 × 100%
```

### 综合回报率
```
房价增值 = 卖出价 - 买入价
总收益 = 房价增值 + (年租金净收入 × 持有年限)
总投资回报率 = 总收益 / 首付款 × 100%
年化回报率 = 总投资回报率 / 持有年限
```

## 风险评分方法

### 现金流风险（40分权重）
- 月供覆盖比 ≥ 1.3：低风险
- 月供覆盖比 0.7-1.3：中风险
- 月供覆盖比 < 0.7：高风险

### 杠杆风险（30分权重）
- 首付比例 ≥ 50%：低风险
- 首付比例 30-50%：中风险
- 首付比例 < 30%：高风险

### 市场风险（30分权重）
- 可承受20%房价下跌：低风险
- 可承受10%房价下跌：中风险
- 无法承受10%房价下跌：高风险

### 综合风险评分
```
综合分数 = 现金流风险 × 0.4 + 杠杆风险 × 0.3 + 市场风险 × 0.3

0-40分：低风险（绿色）
41-70分：中风险（黄色）
71-100分：高风险（红色）
```

**计算结果：**

- 综合回报率：约95%
- 年化收益率：约14.3%
